## Setup

The fastest way to add your server is via QR code from the web UI:

1. In a browser, navigate to your Django Files site and go to **User Settings** > **Clients**.
2. Click **View QR Authentication Code**.
   - On your iPhone: tap the QR code — it will open the app and add the server automatically.
   - On a computer: scan the QR code with your iPhone camera.
3. Tap **Add Server** — done!

## Features

- Share any file (or multiple) from the iOS share sheet and automatically copy the URL to clipboard.
- Add multiple servers and switch between them from the server list.
- Upload directly from your camera or stream your screen/camera live via RTMP.
- Deep links from the web UI open files, albums, and streams directly in the app.
- Fully native iOS UI.
